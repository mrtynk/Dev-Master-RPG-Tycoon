👨‍💻 Dev-Master RPG: Tycoon Edition
Bu proje, bir yazılımcının stajyerlikten global bir şirket CEO'luğuna uzanan kariyer yolculuğunu konu alan, React Native ile geliştirilmiş interaktif bir mobil simülasyon oyunudur. Kullanıcılar teknik soruları çözerek kendilerini geliştirir, kazandıkları parayla donanımlarını toplar ve kendi yazılım ekiplerini kurarlar.

🎯 Oyunun Temel Amacı
Uygulama, öğrenmeyi oyunlaştırma (gamification) dinamikleriyle birleştirir. Sadece kod yazmak yetmez; enerji yönetimi, stratejik donanım alımları ve ekip liderliği gibi yetenekleri de konuşturmanız gerekir.

🚀 Öne Çıkan Özellikler
İnteraktif Soru Bankası: HTML, CSS, JS ve React gibi konularda kolaydan zora 20+ teknik soru.

PC Building Simulator: Kasa, Monitör, Klavye, Mouse ve Kulaklık parçalarını toplayarak donanımınızı tamamlayın.

Kariyer Basamakları: Intern'den başlayıp CTO ve Global CEO unvanlarına kadar yükselin.

Pasif Gelir Sistemi (Idle): Yazılım ekibinizi kurun; siz uygulama içinde olmasanız bile çalışanlarınız sizin için dolar kazansın!

Dinamik UI: Seviye barı, enerji takibi ve her seviyeye uygun profesyonel dashboard tasarımı.

🛠️ Kullanılan Teknolojiler
Framework: React Native (Expo)

State Management: React Hooks (useState, useEffect)

Tasarım: Expo Linear Gradient & Custom StyleSheet

Build: EAS (Expo Application Services)
